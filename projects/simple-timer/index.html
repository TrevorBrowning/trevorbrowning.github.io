⏱️ Simple Timer

Track your time effortlessly. This minimalist timer app was built with Python and Tkinter to help you stay focused and organized — whether you’re tracking coding sprints, writing sessions, or daily habits.

🛠️ Features

  • Start, pause, and reset the timer
  • Keyboard shortcut support for quick use
  • Styled UI with clean layout
  • Exports as a standalone `.exe` for Windows
